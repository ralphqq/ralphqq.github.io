Projects

Here are the links to and description of my project repositories on GitHub:

Data Analysis/Statistics

These repositories house the complete code (mostly Jupyter notebooks) and datasets used in my data/statistical analysis projects.

  • inflation_by_region: Using geopandas to map the June 2016 inflation report.
  • machine-learning-exercises: A collection of Jupyter notebooks tackling exercises from Prof. Andrew Ng's machine learning class.
  • ph-regions-2015-notebook: Generating up-to-date PH regional shape files that include addition of NIR and changes in boundaries.
  • sona-content-notebook: Charting PH history through past SONA content.

Data Collection/Scraping

These include spiders, scrapers, and bots I've created for gathering data from different online sources.

  • bond-yields-downloader: A Python script for downloading daily PDS government securities benchmark rate reports.
  • lotto-results-downloader: Using Selenium and PhantomJS to download historical PCSO Lotto draw results.
  • ph_sona_scrape: A simple scrapy spider for extracting PH presidents' SONA speeches.
  • ph_sona_topics_scraper: A scraper using selenium and PhantomJS
  • psa-reports-downloader: A crawler powered by requests and lxml for downloading attachments from Philippine Statistics Authority reports/releases.
  • trending_business_news_scraper: A spider that uses scrapy, selenium, and PhantomJS to extract trending business news topics and articles from Google Trends.

Web Apps/Pages

These are some of my projects for the Web.

  • pcso_luckypick: An unofficial PCSO Lucky Pick generator web app built with Flask.
  • ralphqq.github.io: The project repo containing the source files for this site, built with Pelican.

Miscellaneous

These are just random stuff I've fiddled with out of curiosity.

  • ArithmoPhile: An arithmetic exercise app built with wxPython.
  • ISSFinder: Tracking the International Space Station on the command line.
  • google-drive-backup: A script for simplifying the task of backing up directories and files into a desktop version of Google Drive.
